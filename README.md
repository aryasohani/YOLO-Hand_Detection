# 🖐️ YOLO Hand Detection

A real-time Hand Detection System built using YOLO and Python.

## 🚀 Features

- Real-time hand detection
- Webcam support
- Image & video detection
- Fast and accurate YOLO model
- Bounding box visualization

## 🛠️ Tech Stack

- Python
- YOLO
- OpenCV
- FastAPI
- JavaScript

## ⚙️ Installation

```bash
git clone https://github.com/aryasohani/YOLO-Hand_Detection.git
cd YOLO-Hand_Detection
```

### Install Requirements

```bash
pip install -r requirements.txt
```

## ▶️ Run Backend

```bash
uvicorn app:app --reload
```

## ▶️ Run Frontend

```bash
npm install
npm run dev
```

- Gesture Recognition
